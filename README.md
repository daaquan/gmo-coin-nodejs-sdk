# GMO Coin Private API SDK for Forex and Cryptocurrency (Node.js/TypeScript)

## Overview
- Typed, minimal-deps client for Forex and Crypto Private REST and WebSocket APIs.
- Supports both Forex (FX) and Cryptocurrency trading.
- Requires Node 18+ (uses built-in `fetch`). Runtime dep: `ws`.

## Background

This project was developed as a Node.js SDK for GMO Coin, providing seamless integration with their trading APIs.

## Purpose

The purpose of this SDK is to provide unified access to FX/Crypto APIs, enabling developers to easily interact with GMO Coin's trading services.

## Key Features

- REST/WebSocket support
- Rate limiting
- Error handling
- Production-ready features
